## CareSync AI – Healthcare Documentation Platform

## Project Overview

CareSync AI is a cloud-based healthcare documentation platform that enables caregivers to record patient notes and securely store structured medical information. The application uses AWS cloud services, serverless architecture, and CI/CD automation to support scalable and secure deployments.

## My Role

QA Engineer | CI/CD Contributor | DevOps Contributor

## Technologies Used

- AWS Lambda
- Amazon API Gateway
- Amazon CloudFront
- Amazon S3
- Amazon Cognito
- Amazon RDS MySQL
- Amazon CloudWatch
- AWS WAF
- Amazon VPC
- GitHub Actions
- Serverless Framework
- TypeScript
- React

## AWS Architecture Components

- Frontend: Amazon S3 + CloudFront
- Authentication: Amazon Cognito
- Backend: AWS Lambda + API Gateway
- Database: Amazon RDS MySQL
- Monitoring: Amazon CloudWatch
- Security: AWS WAF + VPC
